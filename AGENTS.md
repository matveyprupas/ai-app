# Git permissions

NEVER run `git commit`, `git push`, `git rebase`, `git reset`, `git merge`, or any other destructive/write git command without the user explicitly asking you to do so in the current message.

You MAY use read-only git commands freely: `git status`, `git log`, `git diff`, `git show`, `git branch`, `git stash list`.
